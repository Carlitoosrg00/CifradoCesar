# CifradoCesar 💻💻


Programa donde se debe solicitar un texto y una clave introducida por el usuario. 

En este programa, cada letra deberá desplazarse X posiciones en el alfabeto por dígito introducido por el usuario, es decir:
  - Si es 2 el número introducido y la palabra es Hola;
    - H -> J
    - O -> Q
    - L -> N
    - A -> C

Si la palabra sobrepasa la letra Z, el programa deberá de volver a empezar el abecedário:
El programa debe respetar aparte los espacios y carácteres especiales sin cifrar.
